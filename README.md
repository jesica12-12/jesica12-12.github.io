# Profil William Shakespeare

## Laporan Praktikum Coding: Profil William Shakespeare

Proyek ini adalah tugas praktikum coding yang bertujuan untuk membuat halaman profil sederhana dan semantik menggunakan dasar-dasar HTML. Halaman ini menyajikan informasi ringkas mengenai biografi, karya terkenal, dan fakta menarik tentang William Shakespeare.

### Deskripsi Proyek
Halaman ini dibuat dengan fokus pada struktur konten yang rapi dan penggunaan elemen HTML yang sesuai untuk penyajian teks dan daftar.

### Fitur Utama Halaman
* **Judul Halaman:** `Laporan Praktikum Coding: Profil William Shakespeare`.
* **Struktur Teks:** Penggunaan tag heading (`<h1>`, `<h2>`) untuk judul utama dan subjudul.
* **Daftar Terstruktur:**
    * Menggunakan **Ordered List (`<ol>`)** untuk daftar Karya-Karya Terkenal.
    * Menggunakan **Unordered List (`<ul>`)** untuk daftar Fakta Menarik.
* **Penekanan Teks:** Menggunakan tag seperti `<strong>`, `<mark>`, dan `<ins>` untuk memberi penekanan pada informasi penting.
* **Visualisasi:** Menyertakan gambar potret William Shakespeare yang dipusatkan di tengah halaman.
* **Penataan Dasar:** Menggunakan atribut dasar HTML (`align`, `bgcolor`) untuk penataan latar belakang dan perataan teks.

### Teknologi yang Digunakan
* **HTML5** (Murni)
